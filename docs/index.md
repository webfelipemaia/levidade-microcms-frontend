# Leve UI Kit

O **Leve UI Kit** é uma biblioteca de design e desenvolvimento para aplicações Vue.js 3. Ele foi concebido para unir a robustez do **Bootstrap 5** com a reatividade e simplicidade da **Composition API**, oferecendo uma base sólida, leve e pronta para produção.

Nossa proposta é fornecer componentes puramente visuais e estruturais, permitindo que você foque na interface e na experiência do usuário (UX) sem a sobrecarga de frameworks complexos ou dependências excessivas de lógica de backend.

---

## O que é um UI Kit?
Um UI Kit (**User Interface Kit**) é uma biblioteca de recursos que contém elementos de design e componentes prontos. Em vez de recriar um botão, um modal ou uma barra de navegação do zero em cada projeto, você utiliza as peças do kit. Isso garante consistência visual, agilidade no desenvolvimento e facilidade de manutenção.

## O que são Componentes?
Componentes são blocos de construção reutilizáveis e independentes. No **Leve UI Kit**, cada elemento (como o `LSidebar` ou o `LNavigation`) é um arquivo `.vue`. Eles encapsulam sua própria estrutura (HTML), lógica (JS/TS) e estilo (CSS/SCSS), permitindo uma arquitetura modular.

## Tecnologias Utilizadas
* **Vue.js 3:** Core do framework utilizando *Composition API*.
* **Vite:** Ferramenta de *build* de última geração para performance extrema.
* **Bootstrap 5:** O padrão da indústria para layouts responsivos e utilitários.
* **Bootstrap Icons:** Biblioteca de ícones leve e versátil.

## Vantagens e Desvantagens

### Vantagens
* **Leveza:** Código limpo, sem bibliotecas pesadas de terceiros além do essencial.
* **Customização:** Fácil ajuste de temas através de variáveis CSS e classes Bootstrap.
* **Desacoplamento:** Frontend 100% focado em interface, independente de APIs.
* **Baixa Curva de Aprendizado:** Familiaridade imediata para quem já conhece Bootstrap.

### Desvantagens
* **Dependência do Bootstrap:** O kit é estruturado em torno do ecossistema Bootstrap.
* **Estilo Focado em Dashboards:** A estética atual é otimizada para sistemas de gestão e painéis administrativos.
